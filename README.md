cinejs
======

Node JS Application for Cinema Management
```sh
  $ git clone git://github.com/madhums/nodejs-express-mongoose-demo.git
  $ npm install
```

Lancer serveur
```sh
  $ nodemon aps.js
```

Lancer le navigateur
 http://localhost:3000/cinema
